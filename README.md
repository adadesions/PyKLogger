## Getting Started

PyKLogger is provide both CLI and GUI for user.

### Prerequisites
CLI 
* Python 3.5+
* Pip 19.1.1+
* Kivy 1.10.1+
* pygame 1.9.6+
* pynput 1.4.2


### Installation and Usage

1. Clone the repo
```sh
git clone https://github.com/adadesions/PyKLogger.git
```
2. Install dependency
```sh
cd PyKLogger
pip install -r requirements.txt
```

3. Running
```python
# GUI
python3 GUI_main.py

# CLI
python3 main.py
```
